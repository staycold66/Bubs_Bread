## Game Overview

In Bubs' Bread Adventure, you control Bubs, a hungry rat terrier on a mission to collect as much bread as possible. The game features smooth animations, retro-style pixel art graphics, and an engaging collecting mechanic.

### Features

- Pixel art graphics with a detailed rat terrier character
- Smooth, responsive controls
- Dynamic bread spawning system
- Animated background patterns
- Score tracking
- Victory celebration when Bubs collects enough bread
- Stylish UI with glowing effects and gradients
